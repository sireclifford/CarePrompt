# CarePrompt

A non-verbal communication aid for residents in long-term care and dementia facilities, built for both care staff and residents to use directly.

## Why this exists

Built by an iOS developer working as a care aide in a long-term care and dementia facility, drawing on direct experience with the communication challenges residents face.

## Status

🚧 Early development — currently building the core data model and Resident Mode tap-to-speak experience.

## Tech Stack

- SwiftUI
- SwiftData
- AVFoundation (text-to-speech)
- CloudKit (planned, for sync)
